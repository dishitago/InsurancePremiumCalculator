**Insurance Premium Calculator

This Java program calculates the insurance premium for buildings based on:

1.Building type (Domestic / Commercial)
2.Amount insured
3.Number of previous claims
->The calculation follows the rules given in the assignment document, including:
1.Different premium rates
2.Fixed processing charges
3.No-claim bonus (if applicable)

**How to Run
javac InsurancePremiumCalculator.java
